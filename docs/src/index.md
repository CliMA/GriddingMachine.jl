# GriddingMachine

Global datasets to feed [Emerald](https://github.com/Yujie-W/Emerald) and [CliMA Land](https://github.com/CliMA/Land).


## Install and use
```julia
using Pkg;
Pkg.add("GriddingMachine");
```
